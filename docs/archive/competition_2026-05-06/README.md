# Workflow Template Competition Archive

這是 2026-05-06 的一次性 trading workflow template 回測競賽證據包。

主要讀法：

- `report.md`：總排名與各 template 判定
- `FINAL_HEAD_TO_HEAD_REPORT.md`：Codex / Claude head-to-head
- `METHODOLOGY.md`：當時的方法與資料輸出說明
- `scripts/`：當時用於產生報告與資料的腳本

注意：此資料夾已從 `docs/COMPETITION/` 移入 archive。部分歷史文件與腳本仍保留當時的 `docs/COMPETITION/...` 路徑字串，若要重跑需先檢查輸出路徑。
