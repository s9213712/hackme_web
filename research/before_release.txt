2. 發布前檢查腳本

新增：
scripts/release_check.py

檢查：
- 是否有預設密碼
- 是否設定 root password
- 是否設定 production secret
- 是否啟用 HTTPS / secure cookie
- storage root 是否安全
- snapshot 是否可建立
- audit chain 是否正常
- integrity guard 是否無 high-risk pending
- secrets scan 是否通過
- DB schema 是否最新
- logs 是否可寫
- SMTP / Discord 若啟用則設定完整
- debug mode 是否關閉

執行：
python3 scripts/release_check.py --strict
