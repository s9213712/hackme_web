任務：實作 hackme_web 上線前 10 個高價值營運功能

總目標：
補齊正式發布網站需要的安裝、通知、安全事件、維護、檔案隔離、資料保留、資料匯出/刪除、雙人覆核、設定版本歷史、版本/migration 狀態能力。

原則：
- 不新增大型社群功能。
- 不破壞既有 RBAC / audit / Integrity Guard / snapshot。
- 所有 root/admin 操作必須寫 audit。
- 所有 secret 不得明文回傳。
- 危險操作需二次確認。
- 優先補 DB migration、service、API、UI、tests、docs。

────────────────────────
1. 首次啟動安裝精靈
────────────────────────

目標：
新部署時不要依賴預設密碼或手動改 DB。

新增：
setup_state
- id INTEGER PRIMARY KEY CHECK(id=1)
- setup_completed INTEGER DEFAULT 0
- completed_at TEXT
- completed_by INTEGER

system_settings：
- setup_required = true

API：
- GET /api/setup/status
- POST /api/setup/complete

流程：
1. 若 DB 無 root 或 setup_completed=0，進入 setup mode。
2. setup 頁面要求：
   - root username
   - root email
   - root password
   - public_base_url
   - storage root
   - log root
   - snapshot root
3. 檢查密碼強度。
4. 建立 root。
5. 關閉 setup_required。
6. 寫 audit：SETUP_COMPLETED。

安全：
- setup complete 後不可再次開放。
- 若已存在 root，setup API 不得覆蓋 root。
- production 禁止 demo password。

UI：
- public/setup.html 或登入頁導向 setup panel。

測試：
- fresh DB 可完成 setup
- setup 完成後不能重跑
- 弱密碼被拒絕

6. 資料保留政策
────────────────────────

目標：
避免 logs/snapshots/deleted files 無限增長。

system_settings：
- retention_logs_days = 90
- retention_security_events_days = 365
- retention_deleted_files_days = 30
- retention_snapshots_days = 30
- retention_audit_logs_days = 0  # 0 = keep forever
- retention_auto_cleanup_enabled = false

新增表：
retention_cleanup_runs
- id
- started_at
- finished_at
- status
- deleted_counts_json
- error_message

Service：
services/retention.py
- compute_cleanup_plan()
- run_cleanup(dry_run=True)
- delete_expired_soft_deleted_files()
- prune_old_logs()
- prune_old_snapshots()

API：
- GET /api/admin/retention/policy
- PUT /api/admin/retention/policy
- POST /api/admin/retention/dry-run
- POST /api/admin/retention/run

安全：
- audit logs 預設永久
- cleanup 前先產生 report
- 真刪除需 confirm="RUN_RETENTION_CLEANUP"

測試：
- dry-run 不刪資料
- confirm 錯誤不執行
- audit logs 不被誤刪
